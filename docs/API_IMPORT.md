# Import OpenAPI to Postman / Apifox

This project includes an OpenAPI file at:
```
docs/openapi.yaml
```

---

## Postman Import

1. Open Postman
2. Click **Import**
3. Select **File**
4. Choose `docs/openapi.yaml`
5. Postman will create a collection

---

## Apifox Import

1. Open Apifox
2. Click **Import**
3. Choose **OpenAPI**
4. Select `docs/openapi.yaml`
5. Finish import and set the base URL

---
